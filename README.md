# Netflix 1990s Movie Analysis

## Project Overview

This project explores Netflix movie data with a focus on films released during the 1990s. The goal was to practice exploratory data analysis (EDA) using Python and pandas while investigating trends in movie duration and action movies from that decade.

## Questions Explored

1. What was the most common movie duration among Netflix movies released in the 1990s?
2. How many action movies released in the 1990s had a duration of less than 90 minutes?

## Tools Used

* Python
* Pandas
* Matplotlib

## Key Findings

* The most common movie duration in the 1990s dataset was calculated using the mode of the duration column.
* The number of short action movies (less than 90 minutes) was identified through filtering and counting operations.
